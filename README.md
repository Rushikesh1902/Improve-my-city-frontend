# [CivicEye](https://improve-my-city-frontend-ten.vercel.app/) 
Civic Eye is a simple web-based platform that allows citizens to report civic issues in their locality such as potholes, electricity problems, water supply issues, and garbage complaints.

## Project:
CivicEye — Report. Track. Resolve.

## Team: Brogrammers

## Suggested APIs
- ``` POST /api/complaints ```— Create complaint (body: category, description, location, photo URL)
- ``` GET /api/complaints ``` — List complaints
- ``` GET /api/complaints/:id ``` — Get complaint details
- ``` PUT /api/complaints/:id/status ``` — Update status (body: status)
- ```GET /api/stats — Basic stats``` (resolved count, avg resolution time)
